## About

This repository documents my learning journey throughout the **Amroha Police Cyber Security Internship Program (APCSIP-2026)**.

APCSIP-2026 is a cybersecurity internship initiative organized by **Amroha Police, Uttar Pradesh**, designed to provide practical exposure to cyber security, cybercrime investigation, cyber forensics, cyber law, digital safety, and modern cyber policing practices.

The objective of this repository is to maintain a structured record of:

* Daily internship reports
* Session notes
* Learning resources
* Practical exercises
* Cybersecurity concepts
* Assignments and activities
* Key takeaways

---

## Program Overview

* **Program:** Amroha Police Cyber Security Internship Program (APCSIP-2026)
* **Mode:** Offline / In-Person
* **Duration:** 11 June 2026 – 22 June 2026
* **Organized By:** Amroha Police, Uttar Pradesh
* **Focus Areas:**

  * Cyber Security Fundamentals
  * Cyber Crime Investigation
  * Digital Forensics
  * Cyber Law
  * OSINT
  * Threat Intelligence
  * Incident Response
  * Cyber Awareness
  * Ethical Hacking Concepts

---

## Learning Goals

Throughout this internship, I aim to:

* Understand cybercrime investigation workflows.
* Learn practical cybersecurity methodologies.
* Explore cyber forensics techniques.
* Improve OSINT and analytical skills.
* Gain exposure to cyber law and law-enforcement perspectives.
* Develop a stronger foundation for offensive security and penetration testing.
